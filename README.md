PROYECTO: ALKE WALLET M2
Fundamentos del Desarrollo Frontend

OBJETIVO DEL PROYECTO
Desarrollar una aplicación web utilizando HTML, CSS y JavaScript que implemente interactividad, manejo de eventos, validación de formularios y actualización dinámica de la información en pantalla.

DESCRIPCIÓN
La aplicación simula una billetera virtual. Permite iniciar sesión, visualizar el saldo, realizar depósitos, enviar dinero a contactos y consultar los últimos movimientos. La información se gestiona mediante JavaScript y se persiste usando LocalStorage.

FUNCIONALIDADES IMPLEMENTADAS

Captura de eventos en formularios y botones

Validación de campos antes de procesar datos

Actualización dinámica del saldo

Registro de transacciones (depósitos y transferencias)

Visualización y filtrado de movimientos

Persistencia de datos con LocalStorage
